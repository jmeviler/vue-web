# vue-web-demo

> vue demo for web

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 目录结构

``` js
├── build                     // 构建相关
├── config                    // 配置相关
├── src                       // 根目录
│   ├── api                   // 请求相关
│   ├── assets                // 静态资源
│   ├── components            // 公共组件
│   ├── constant              // 常量文件
│   ├── modules               // 模块相关
│   ├── router                // 路由相关
│   ├── store                 // store管理
│   ├── scss                  // 全局样式
│   └── utils                 // 公共方法
├── static                    // 第三方资源
├── .babelrc                  // babel-loader 配置
├── eslintrc.js               // eslint 配置项
├── .gitignore                // git 忽略项
├── index.html                // html模板
└── package.json              // package.json
```
